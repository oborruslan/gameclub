# GALAXY VR ARENA Website

Site static, responsive, fără dependențe. Deschide `index.html` într-un browser sau publică întregul folder pe orice serviciu de hosting static.

Pagina folosește trei fotografii originale: Hero, PlayStation Zone și VR Arena. Celelalte panouri vizuale sunt construite din CSS.

## Înainte de publicare

Înlocuiește în `index.html`:

- `[ADRESA COMPLETĂ]`
- `[NUMĂR DE TELEFON]`
- `[PROGRAM]`

Confirmă tarifele pentru locația din Ungheni și conectează formularul din `script.js` la WhatsApp, email sau un serviciu de rezervări. În versiunea curentă, formularul este validat local și nu transmite date.
